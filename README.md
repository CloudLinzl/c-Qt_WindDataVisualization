# 测风数据分析与可视化软件平台

基于C++和Qt开发的激光测风雷达数据分析与可视化应用程序。

## 已完成功能

1. **数据读取与时间对齐**

   - 读取雷达测风数据和云台角度数据文件
   - 按时间戳对齐两个数据源
   - 注意：xlsx文件需要先通过xlsx_to_csv.py转换为CSV文件

2. **数据预处理与SNR阈值过滤**

   - 离群值检测与修复
   - NaN值填充
   - 可配置的SNR阈值过滤

3. **图形界面**

   - 菜单栏：数据文件选择、结果导出、视图切换
   - 控制面板：SNR阈值输入、距离范围选择、时间滑块、模式切换
   - 数据可视化区域：PPI扫描热力图、距离-风速/SNR折线图
   - 状态栏：文件名、扫描模式、数据点统计

4. **PPI扫描风速实时可视化**
   - PPI扫描热力图（极坐标显示）
   - 支持距离范围选择
   - 彩色热力图显示径向风速
   - 低SNR数据灰色显示
   - 实时更新可视化

   - 距离-风速剖面图
   - 距离-SNR剖面图

5. **湍流强度 (TI) 分析**
   - 实时计算湍流强度（滑动窗口法：支持调整窗口大小）
   - 支持一键切换显示风速/湍流强度

6. **交互式可视化与视图控制**

   - 支持鼠标滚轮缩放、左键拖动平移、鼠标悬停显示详细信息（方位角、距离、数值）

1. **高级导出功能**
   - **导出数据**：导出预处理后的完整数据，自动添加UTF-8 BOM头。
   - **导出图像**：一键导出包含PPI热力图和底部折线图的组合图像，以时间戳命名。

## 编译与运行

### 系统要求

- C++17 或更高版本
- CMake 3.16 或更高版本
- Qt5 或 Qt6

## 使用说明

### 加载数据

1. 点击菜单栏 **文件 → 打开风速数据**，选择风速CSV文件
2. 点击菜单栏 **文件 → 打开云台角度数据**，选择云台角度CSV文件
3. 点击 **加载数据** 按钮或使用菜单 **文件 → 加载数据**（快捷键Ctrl+L）

### 调节参数与交互

- **SNR阈值**：调整信噪比阈值，低于阈值的数据将显示为灰色
- **最小/最大距离**：设置可视化的距离范围
- **时间滑块**：拖动滑块查看不同时刻的数据
- **显示模式**：在工具栏选择“风速”或“湍流强度”模式
- **缩放/平移**：在PPI图上滚动鼠标滚轮进行缩放，按住左键拖动进行平移
- **查看详情**：鼠标悬停在PPI图上，左上角将显示当前位置的详细信息

### 导出功能

- **导出数据**：将预处理后的完整数据导出为CSV文件
- **导出图像**：将当前时刻的PPI热力图和剖面图组合保存为PNG图片，默认文件名为当前数据时间

## 技术说明

### PPI扫描可视化

- PPI图以雷达为中心，使用极坐标系统
- 半径表示距离，角度表示方位角
- 方位角0°对应东偏南15度，顺时针扫描
- 仰角范围5°-20°，每5°一个扫描层
- **缓存优化**：使用 `QPixmap` 缓存机制，大幅提升缩放和平移时的渲染性能

### 数据预处理算法

1. **离群值检测**：比较相邻距离门的风速值，差异超过阈值则视为离群值
2. **异常值修复**：使用前后邻居平均值填充离群值和NaN值
3. **SNR过滤**：根据用户设定的SNR阈值过滤低信噪比数据

### 颜色映射

热力图使用彩虹色谱映射风速值：
- 蓝色：低风速 / 低湍流
- 青色：中低风速
- 绿色：中等风速 / 中等湍流
- 黄色：中高风速
- 红色：高风速 / 高湍流
